Introduction

This document describes the various repositories included by default in the most recent release. For information on how to manage these repos, see Synaptic.

Validity: MX-14.2, released June 30, 2014

Default repositories

Sources of software packages are listed in 4 files contained in /etc/apt/sources.list.d/. Note that users may see minor changes due to localization (e.g., http://ftp.uk.debian.org/debian/ instead of http://ftp.us.debian.org/debian/).

antix.list

# Use with Debian Wheezy repository. Set as default for MX-14
deb http://antix.daveserver.info/stable/ stable main

debian.list

# Debian Stable
deb http://ftp.us.debian.org/debian/ wheezy main contrib non-free
deb http://security.debian.org/ wheezy/updates main contrib non-free
# deb-src http://ftp.us.debian.org/debian/ wheezy main contrib non-free
# Wheezy backports
deb http://ftp.us.debian.org/debian/ wheezy-backports main contrib non-free
# Multimedia Stable
# deb http://www.deb-multimedia.org/ wheezy main non-free
