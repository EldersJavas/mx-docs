Introduction

The default browser for MX-14 is QupZilla, a modern web browser based on WebKit core and Qt Framework. This fast and lightweight application is pretty self-explanatory to use, but here are some basic items.

  • Set it up as you want it via Edit > Preferences.
  • The default search engine is set to be DuckDuckGo for security reasons, but you can click the left side of the search box to switch, add or delete engines.
  • Adobe Flash is installed by default; a known issue is that it sometimes crashes on first use after installation, then never again. To manage, use MX Flash Manager
  • AdBlock and a couple of other integrated extended functions are available by clicking Tools.
  • Click Tools > New Private Window to avoid storage of history, cookies, etc. during a single session.

Tips and tricks

Some features are not obvious at first

  • You can increase visible space by clicking View > Toolbars, and unchecking Menu, Navigation and/or Bookmarks toolbar. A menu icon then appears at the right end of the navigation bar.
  • The first time a link is clicked with Flash content, QupZilla sometimes crashes; but it never seems to happen again.
  • RC any tab to see a rich menu about tab management.
  • Click Tools > Clear Recent History to increase security for a closed session.
  • If you have trouble with a particular site, try clicking Edit > Preferences > Other > Change browser identification. Click on User Agent Manager, and either change the identification for the site or make a global identification change.
  • If you install a different browser, here are a couple of things to think about.
    • You can import the default QupZilla bookmarks from /usr/local/share/mx/DefaultBookmarks.html.
    • You need to identify your preferred browser using Xfce4, not the browser itself, so that links for html documents will work. Click the Settings manager > Preferred Applications, and identify the browser you want to use.
