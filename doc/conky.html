Introduction

Conky is a program originally designed for use as a system monitor that can also gather and display almost any kind of information.

Installation

Follow these steps to install the Conky program and manager.

  • Open Synaptic and click Settings > Repositories.
  • Check the box to enable this repo: http://main.mepis-deb.org/mepiscr/testrepo/
  • Close the Repositories window, then click the Reload button, and wait for it to finish.
  • Click the search icon (magnifying glass) on the menu bar and type conky-manager. Then click Search.
  • When it is finished, mark conky-manager for installation.
  • A small window will open up stating that you also need to download additional files with them. Click Mark.
  • Click Apply on the menu bar.
  • After installation has completed, reopen Settings > Repositories and uncheck the .../testrepo box. Then close and exit Synaptic.

Configuration

  • Click Start menu > Accessories to find Conky Manager.
  • In Conky Manager, you can highlight each conky and hit Preview to see what it looks like. Be sure to close each preview before going to another.
  • It may seem that Conky Manager has frozen, but that is just because there is a hidden window with the preview hiding behind it.
  • When you have decided which to use, check the box to select it. It will be auto-installed.
  • The configuration files are stored in the folder ~/conky-manager
